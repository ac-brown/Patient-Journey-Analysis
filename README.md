# Patient Journey Analysis

End-to-end healthcare analytics project using SQL and Tableau to identify cost drivers, readmission risk, patient satisfaction trends, and operational inefficiencies.

# Tools Used
- SQLite (SQL analysis)
- Tableau (visualization)
- GitHub (documentation & portfolio)

# Project Overview
This project analyzes 3,000 simulated hospital patient encounters to evaluate:

- Cost of care drivers
- 30-day readmission risk
- Patient satisfaction factors
- Operational efficiency
- Admission trends over time

SQL was used to generate analytical summaries, and Tableau dashboards were built to visualize key insights.
